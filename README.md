# TodoApi-ASP.NetCore-MinimalAPI
TodoApi, minimal API using ASP.NET Core.
<br>

Tutorial<a href="https://learn.microsoft.com/en-us/aspnet/core/tutorials/min-web-api?view=aspnetcore-8.0&tabs=visual-studio"></a>
<br>

Notes:
<ul>
  <li>Minimal APIs are architected to create HTTP APIs with minimal dependencies. They're ideal for microservices and apps that want to include only the minimum files, features, and dependencies in ASP.NET Core.</li>
  <li>To enable <code>builder.Services.AddDatabaseDeveloperPageExceptionFilter();</code> install the 
  <code>Microsoft.AspNetCore.Diagnostics.EntityFrameworkCore</code> NuGet package.</li>
</ul>
